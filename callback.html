📌 Próximos passos — Trocar o code pelo access token

  1. Copie o Código de Autorização acima.
  2. Faça um POST para https://api.mercadolibre.com/oauth/token com os parâmetros:
    grant_type=authorization_code, client_id, client_secret, code, redirect_uri
  3. Você receberá o access_token e o refresh_token na resposta.
  4. O access token expira em 6 horas. Use o refresh_token para renová-lo.
